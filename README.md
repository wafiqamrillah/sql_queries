# SQL Query Cheatsheet

Proyek ini berisi kumpulan query SQL yang disusun seperti sebuah cheatsheet berdasarkan studi kasus yang pernah dibuat sebelumnya. Setiap query di dalam repositori ini diharapkan dapat membantu dalam menyelesaikan berbagai permasalahan yang sering ditemui dalam pengelolaan database.

## Daftar Program SQL yang Didukung

- SQL Anywhere 17

## Studi Kasus yang Terlibat

- [Monitor ERP - Customized Reports modules]

## Struktur Proyek

Setiap folder atau file di dalam repositori ini berisi query-query yang dikategorikan berdasarkan jenis kasus atau kebutuhan tertentu, sehingga memudahkan pencarian dan penggunaan ulang.
